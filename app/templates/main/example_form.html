DISCO Data Prep

How to use

The DISCO application is a paid for tool the allows users show user journeys of users moving through GOV.UK

The data that needs to be input into the tool needs to be formated in a particular way that can be used by the tool

The DISCO Data Prep tool can be used to provide the data you need in the correct format

Simply fill in each field and click on the "Continue" button

The tool connects to Bigquery so please be mindful of the date range when extracting the data you need Try and keep the time period to one day

You have the ability to download the results into a csv from the results page
